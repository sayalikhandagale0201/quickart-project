Quickart is a full-stack E-Commerce website built using Next.js, Spring Boot, and PostgreSQL. It provides secure authentication, product management, cart functionality, and order processing.

Tech Stack

Frontend: Next.js

Backend: Spring Boot (REST APIs, JWT Authentication)

Database: PostgreSQL

 Features

User Registration & Login

Role-Based Access (Admin/User)

Product Listing

Add to Cart

Order Placement

Secure REST APIs

A modern, scalable full-stack e-commerce solution.
